### 3d-models-constructor

---
![alt](./Constructor.gif)

---
Constructor of my alternative 3d models for the board game warhammer 40k.
Helps to imagine how the model will look with different options. Written with Three.js

---

###### Technologies

- React 18.2.0
- React-redux 8.0.2
- React-spinners 0.13.3
- Mui 5.9.2
- Three 0.141.0

---

For start:
```
cd client
npm install
npm start
```
